Data Flow
FilterBar (UI) → useFilters (shared state) → getCurrentFilters() → api.js (Axios) → FastAPI endpoint → apply_filters() + filter_by_month() → JSON data → Pydantic model → JSON response → Vue ref() → re-render
Filter sentinel: "all" skips that filter. Month supports ISO format (2025-09) and quarter format (Q3-2025). Inventory endpoints do not support month filtering.
